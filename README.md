# nanarinote-flutter
 
基于 vitepress@0.22.3 构建的跨平台APP开发的学习笔记，包含

- dart
- Flutter


## vite-setup

安装依赖（若没有yarn需要先全局安装）

```bash
yarn install
```

启动命令（或运行 `docs_dev.cmd` ）：

```bash
# serve with hot reload at localhost:3000
yarn docs:dev
```
